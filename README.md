# Ecommerce SQL Analysis

## Project Overview

This project analyzes ecommerce sales data using SQLite, SQL, Pandas, and Jupyter Notebook.

## Objectives

* Calculate total revenue
* Identify top customers
* Analyze revenue by city
* Identify repeat customers
* Calculate average order value

## Tools Used

* Python
* SQLite
* Pandas
* Jupyter Notebook

## Key Findings

* Total Revenue: ₹41,000
* Top Customer: Customer 103
* Top City: Pune
* Repeat Customers: 3
* Average Order Value: ₹5,125

## Business Recommendations

* Focus marketing efforts in Pune.
* Retain high-value customers.
* Increase repeat purchases from one-time customers.
* Monitor customer lifetime value.

## Skills Demonstrated

* SQL Joins
* GROUP BY
* HAVING
* Aggregations
* Business Analysis
* Data Interpretation
